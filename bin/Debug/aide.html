My Super Calculator - L'outil ultime pour tous vos besoins de calcul
Nom de la Société : Institut Supérieur de Génie Informatique
Promotion 2ème année de l'année universitaire 2022-2023

Description


Cette application est réalisée par les étudiants de l'Institut Supérieur de Génie Informatique pour mettre en pratique ses connaissances en programmation Dot Net avec le langage Visual Basic.Net.

Aide


L'utilisation de My Super Calculator est très simple, en cliquant sur les différents boutons proposés, elle répond aux opérations standard de n'importe quelle calculatrice, en outre, cette 1ère version permet de convertir les degrés de température d'une unité à l'autre. Cette version permet aussi de garder une trace de toutes les opérations effectuées.
